## Cài đặt

```bash
bun add -g @better-update/cli
```

## Ship một update

```bash
better-update login
better-update init
better-update update publish --branch main --message "Fix login screen"
```

Device trên channel khớp pull bundle mới ở lần launch tiếp theo.

## Rollout từ từ, hoặc revert một câu lệnh

```bash
better-update update rollout set <updateId> 10
better-update update rollout complete <updateId>

better-update update rollback --branch main
```
